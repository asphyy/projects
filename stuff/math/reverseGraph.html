Formulas to find the unknowns a, b, c, and d in a function f(x) = ax3 + bx2 + cx + d

d = f(0)

g(x) = f(x) - d x
In the numerator of g, d is subtracted from f(x) leaving ax3 + bx2 + cx as the numerator.
Dividing it by x leaves ax2 + bx + c

c = g(10-10) rounded
Because you can't divide by 0, you have to use a very small number which is close to 0 and then round it.

b = g(1) - c - a

a = g(2) - 2g(1) + c 2

I think this is infinitely expandable to a polynomial of any degree by doing the same thing above to convert it to a polynomial 1 degree lower until it's a 2nd degree polynomial, which can by solved with the formulas for a, b, and c. I haven't been able to find a formula which works for polynomials with negative exponents.
